# 🛒 S-Mall 企业级商业空间引擎 (v3.0.0)

欢迎来到 Space² 元宇宙的终极商业操作系统。本插件将传统的扁平电商彻底升维，打造具备毫米级坐标锚定、物理法则缩放以及绝对防伪溯源能力的“全息主权展示间”。

## ⚠️ S2 生态前置依赖声明 (安装必读)
S-Mall 企业版是建立在 Space² 基建之上的高层应用。在 Openclaw 部署本插件前，**必须**确保环境已集成以下底层核心：

1. **空间产权基石：`s2-sssu-spatial-genesis`**
   本插件不负责凭空创造空间。您必须先通过 Genesis 插件获取合法的 SUNS v3.0 地址编码（数字地契），才能在此基础上划分 L5 业务楼层与 L6 橱窗。
2. **物理环境网桥：`s2-space-agent-os-mothership`**
   在进行 X-SSSU 弹性展示或 1v1 洽谈时，本插件需要调用 Mothership OS 来调动现实/虚拟世界中的“光、声、气”等六要素实施库，烘托极致的交易氛围。
3. **环境变量**：
   需在宿主机配置 `S2_DID_VERIFICATION_ENABLED=true`，以确保在签署智能合约时，能对买卖双方的 22 位法定数字身份证 (S2-DID) 进行防伪鉴权。

## 🌟 核心引擎功能

* **GEO 毫米级降维寻址**：彻底抛弃 URL。采用严密的 6 段式空间坐标 + XYZ 毫米级定位（如 `QIANJIA9-5-1-1000-1000-1200` 代表千家网第5层，1号展示间正中心）。
* **X-SSSU 扩展演示引擎**：打破 2x2x2.4m 的限制。支持等比例放大（客户步入体验巨型设备）或缩小（上帝视角观察微观产品）。
* **SMP 1v1 交易互斥锁**：顾客跨入原点瞬间锁闭空间，屏蔽一切外部干扰，确保高定交易的私密性与独占性。
* **空间烙印智能合约**：交易达成时，将时间戳、毫米级坐标、买卖双方 DID 与当前环境哈希值进行 SHA256 不可逆加密，并连同四大数据流（访客、智能体、产品、环境）存入 1 年期全息审计数据库。
